Guía de instalación de Python, Numpy, Jupyter Notebook y otros

Python

Versiones de Python Actualmente hay dos versiones de python, la 2.x (al momento de escribir esto, la última es la 2.7) y la 3.x (la última es la 3.6). Las versiones 3.x han introducido cambios que hacen que el código no sea compatible con las versiones 2.x.

Para este curso, todo el código utilizará la versión de python 3.5 o 3.6.

Instalación de Python

Windows

Si estás en Windows, recomendamos instalar Anaconda, que ya tiene todos los paquetes necesarios instalados.

Linux

En Linux también se puede utilizar Anaconda, pero seguramente es mejor utilizar el python que te provea tu distribución, e instalar mediante pip los paquetes necesarios con el comando: pip install jupyter matplotlib numpy scikit-image numba

Verificar la instalación de Python

Podés verificar qué versión de Python tenés instalado ejecutando python --version en la consola. Debería decir algo como: 

Python 3.4.3 (default, Nov 17 2016, 01:08:31)
[GCC 4.8.4] on linux
Type "help", "copyright", "credits" or "license" for more information.

Si tienes instalado tanto Python 2.x como Python 3.x, probá utilizando python3 en lugar de python.

Para verificar que los paquetes necesarios estén instalados, ejecuta python (o python3).

Cuadernos de Jupyter

El servidor de cuadernos se inicia ejecutando jupyter notebook desde la línea de comandos. Antes de correr ese comando debemos ir al directorio con los cuadernos, de modo de poder ejecutarlos. El servidor corre continuamente mientras usamos los cuadernos.
