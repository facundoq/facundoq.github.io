To automatically compile ejercicios.md and videos.md to ejercicios.html and videos.html run
the script watch_md.jl with "julia watch_md.jl" in the current folder.

Notes:
1) You'll need Julia installed and in your path (ie, apt-get install julia in debian systems)
2) The script watches for changes in the .md files and automatically regenerates the html file.


You can install an autoreload plugin for firefox/chrome to automatically reload the website whenever the html file changes.
(https://addons.mozilla.org/en-US/firefox/addon/auto-reload/,https://chrome.google.com/webstore/detail/livepage/pilnojpmdoofaelbinaeodfpjheijkbh?hl=en)
